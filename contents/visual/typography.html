Style Usage

Display

Display style is recommeded to be used on in-content pages for delivering primary messages to users.

Title

Title style is used on top as a summary of the following content. A good pair of titles can improve visual readibility of a long content.

Emphasized style

Use emphasized style when there is a part of content that needs to stand out from regular content.

Font Pairing

A set of font styles are recommended to be used primarily, but they can be combined with other font styles.

Color

Choose text color based on purpose. Primary for titles and body text. Secondary for accompanying text that gives further information, details or features to the primary content such as caption text and subtitle of menus.
