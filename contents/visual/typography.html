Typography

Good typography is invisible. People should have the most seamless experience while they're using Firefox products.

Typefaces

In Firefox products we use the standard typeface for each platform.

Windows

Segoe UI is the typeface used on the Windows platform.

macOS

San Francisco is the standard typeface on the Apple's products.

Linux

Ubuntu is the typeface used in the most common Linux distro, Ubuntu.

Web products

Fira Sans is the typeface used on the Firefox products on the web.

Despite Open Sans being use on some Mozilla web properties it is currently not recommended to be used in Firefox products.

Android

Roboto is the standard typeface on Android. Noto is the standard typeface for all languages on Chrome and Android for all languages not covered by Roboto.

iOS

San Francisco is the standard typeface on the Apple's products.

Styles

Scale

The type scale enables you to set the hierarchy of your content. These sizes and styles are designed to balance content density and increase readibility.

Desktop

Emphasis

If two texts feature the same scale use a bolder font weight 600 to emphasize the most important information.

Color

Choose text color based on purpose. Primary color is for main titles and body text.

Usage

Display

Display style is recommeded to be used on in-content pages for delivering primary messages to users.

Title

Title style is used on top as a summary of the following content. A good pair of titles can improve visual readibility of a long content.

Font Pairing

A set of font styles are recommended to be used primarily, but they can be combined with other font styles.
